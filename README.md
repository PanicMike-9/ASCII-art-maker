# ASCII Canvas

- Each pixel is a character
- Arranged in rows and columns
- We can
 * clear it
 * Modify positions
 * print it

## Representing a Canvas

- Two dimensions

``` C
canvas
  |-- row 0: [][][][]
  |-- row 1: [][][][]
  |-- row 2: [][][][]
```
- Each cell stores one character.
- `' '` as empty
- fixed width
- fixed height
